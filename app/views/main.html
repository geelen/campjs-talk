Philosophy

Angular

  • is a framework for building web applications.
  • is an extension to HTML and browsers.
  • asks "What would HTML look like if it was designed for applications, not documents?"
  • asks "How would the browser behave in that case?"
  • is an alternative to Backbone / Ember / Knockout.
  • is an alternative to jQuery.
  • is something new.

Controllers

<div ng-controller='MyCtrl' ...
var MyCtrl = function($scope, ...
  • Scoped to a part of the DOM
  • Should NEVER touch the DOM
  • Instead, it sets values/methods on $scope

$scope

  • This is a view-model
  • It's a plain-old-javascript-object
  • Change it and the DOM changes, like magic*
* causes 90% of the 'WHOA' and 'ANGULAAARRRR' stages
